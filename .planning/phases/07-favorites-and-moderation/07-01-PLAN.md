  <action>
1. In `lib/supabase/types.ts`, add a `favorites` table definition to `Database.public.Tables` following the same pattern as `reviews`:
   ```
   favorites: {
     Row: { id: string; user_id: string; stay_id: string; created_at: string | null }
     Insert: { id?: string; user_id: string; stay_id: string; created_at?: string | null }
     Update: { id?: string; user_id?: string; stay_id?: string; created_at?: string | null }
     Relationships: [
       { foreignKeyName: "favorites_stay_id_fkey"; columns: ["stay_id"]; isOneToOne: false; referencedRelation: "stays"; referencedColumns: ["id"] }
     ]
   }
   ```
   Note: The actual Supabase table must be created manually by the user. The type definition enables TypeScript support.

2. Create `lib/actions/favorites.ts` with 'use server' directive:

   - `toggleFavorite(stayId: string)`: Get session via `supabase.auth.getSession()`. If not authenticated, return `{ error: 'Not authenticated' }`. Check if favorite exists with `.from('favorites').select('id').eq('user_id', user.id).eq('stay_id', stayId).single()`. If exists, delete it. If not, insert `{ user_id: user.id, stay_id: stayId }`. Call `revalidatePath('/', 'layout')` to refresh all pages. Return `{ favorited: boolean }`.

   - `getFavoriteStayIds()`: Get session. If not authenticated, return empty array. Query `.from('favorites').select('stay_id').eq('user_id', user.id)`. Return array of stay_id strings.

   - `getFavoriteStays()`: Get session. If not authenticated, return empty array. Query favorites for user, then query stays matching those stay_ids. Compute avg_rating and review_count the same way `getStays` does (fetch reviews, build ratingMap). Return `StayCard[]`.
  </action>
