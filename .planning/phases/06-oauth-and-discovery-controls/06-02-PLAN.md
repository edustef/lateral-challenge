Update the toolbar `div` in `app/(main)/page.tsx` to be sticky. Change the toolbar row from:

```tsx
<div className="flex items-center gap-3">
```

To:

```tsx
<div className="sticky top-0 z-30 -mx-4 flex items-center gap-3 bg-bg-page px-4 py-3 sm:-mx-6 sm:px-6 lg:-mx-8 lg:px-8">
```

Key classes explained:
- `sticky top-0` — sticks to top of viewport when scrolling
- `z-30` — stays above stay cards (which may have z-10/z-20 for hover states)
- `-mx-4 px-4` (and sm/lg variants) — negative margins + matching padding to make the sticky bar extend full width of the container, matching the parent layout's padding
- `bg-bg-page` — solid background so content doesn't show through when scrolling
- `py-3` — vertical padding for visual breathing room when stuck

Also add a subtle bottom border that appears. Add `border-b border-border/50` to the className to provide visual separation from content below:

Final className:
```
sticky top-0 z-30 -mx-4 flex items-center gap-3 border-b border-border/50 bg-bg-page px-4 py-3 sm:-mx-6 sm:px-6 lg:-mx-8 lg:px-8
```

This is a Tailwind-only change. No JavaScript, no new components, no state management needed.
