1. **lib/utils/price.ts** — Price calculation utility:
   - Export `calculateTotal({ pricePerNight, nights, cleaningFee, serviceFee }: { pricePerNight: number; nights: number; cleaningFee: number; serviceFee: number }): number` — returns total in cents
   - Export `calculateNights(checkIn: Date, checkOut: Date): number` — returns number of nights (minimum 1)
   - Export `formatPrice(cents: number): string` — returns formatted price string like "$150" (reuse pattern from booking-sidebar)
   - All prices are in cents (integers)

2. **lib/actions/bookings.ts** — Server action to create a booking:
   - 'use server'
   - Export `createBooking(formData: FormData): Promise<{ bookingId?: string; error?: string }>`
   - Extract from formData: stayId, checkIn, checkOut, guests, contactName, contactEmail, contactPhone, totalPrice
   - Get authenticated user via `supabase.auth.getUser()` — if no user, return `{ error: 'Not authenticated' }`
   - Validate: checkOut > checkIn, guests >= 1
   - Insert into bookings table: `supabase.from('bookings').insert({ stay_id, user_id: user.id, check_in: checkIn (ISO date string), check_out: checkOut (ISO date string), guests, total_price: totalPrice (number), contact_name, contact_email, contact_phone }).select('id').single()`
   - On success: redirect to `/stays/{slug}/book/confirmation?id={bookingId}` (pass slug via formData too)
   - On DB error (e.g., overlapping dates exclusion constraint): return `{ error: 'These dates are no longer available' }`

3. **app/(main)/stays/[slug]/book/page.tsx** — Checkout page (server component):
   - Accept `params` prop with slug
   - Fetch stay using `getStayBySlug(slug)` — if not found, call `notFound()`
   - Import and render `<CheckoutForm stay={stay} />` (client component, built in Task 2)
   - Page title: "Book {stay.title}"
   - Include a back link to `/stays/{slug}`
   - Auth guard is handled by proxy.ts (redirects to login), but this page can also check user as a safety net
