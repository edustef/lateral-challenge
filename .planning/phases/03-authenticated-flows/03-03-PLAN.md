   - Export `createReview(formData: FormData): Promise<{ error?: string; success?: boolean }>`

   - Get authenticated user via `supabase.auth.getUser()` — if no user, return `{ error: 'You must be signed in to leave a review' }`

   - Submit button: "Submit review" with accent styling
   - Use useActionState (from 'react') or useTransition to handle form submission state:
     - Pending: disable button, show "Submitting..."
     - Success: show green "Review submitted!" message, reset form
     - Error: show red error message
   - Hidden input for stayId
   - Style with design tokens
