## Critical Issues (Blockers)

### 1. CRITICAL: proxy.ts must be renamed to middleware.ts
**Affected:** AUTH-03, CHKT-08, TEST-02
**Impact:** Session refresh never happens, checkout auth guard never fires
**Fix:** `mv proxy.ts middleware.ts` — file contents are correct

### 2. HIGH: OTP redirect chain broken
**Affected:** CHKT-08
**Impact:** After OTP login from checkout redirect, user lands on / instead of /stays/[slug]/book
**Fix:** Login page must read `?redirect=` from URL, pass through `loginWithOtp` → `emailRedirectTo` → auth callback

### 3. MEDIUM: Search bar price display
**Affected:** DISC-05
**Impact:** Search preview shows €15000 instead of $150.00
**Fix:** Use `formatPrice()` in search-bar.tsx line 147
